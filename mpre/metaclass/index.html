Docstring

A descriptor object used by the Documented metaclass. Augments instance docstrings with introspected information

Documented

A metaclass that uses the Docstring object to supply abundant documentation for classes

Instance_Tracker

Provides instance tracking and counting attributes.

    Note as of 3/3/2015: the class must implement these attributes,
    it is not performed by this metaclass

Metaclass

A metaclass that applies other metaclasses. Each metaclass in the list Metaclass.metaclasses will be chained into a new single inheritance metaclass that utilizes each entry. The methods insert_metaclass and remove_metaclass may be used to alter the contents of this list.

    Implementation currently under examination due to compiling with
    cython being broken

Method_Hook

Provides a hook on all methods for the new class. This metaclass uses this hook to wrap each method in a Runtime_Decorator.
